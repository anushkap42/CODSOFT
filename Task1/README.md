# Task 1: Rule-Based Chatbot

This project is a simple rule-based chatbot created using Python.

The chatbot responds to user inputs using predefined rules with if-else statements.

## Features
- Responds to greetings like hello and hi
- Answers simple questions
- Ends the chat when the user types "bye"

## Technology Used
Python

## How to Run

1. Install Python
2. Download the chatbot.py file
3. Open terminal or command prompt
4. Run the command:

python chatbot.py
