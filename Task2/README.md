# Task 2: Tic Tac Toe AI

This project implements a Tic Tac Toe game where a human player competes against an AI.

## Algorithm Used
Minimax Algorithm

## Features
- Human vs AI gameplay
- AI never loses
- Command line interface

## How to Run

1. Install Python
2. Download tictactoe.py
3. Run:

python tictactoe.py
